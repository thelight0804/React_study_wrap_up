## 1. 勉強会の目的
- **目的** : フロントエンドでよく使うフレームワークであるReactを学び、自分だけのプロジェクトを作ります
- **期間** : 2023年 9月 11日 ~ 2024年 1月 21日
<br>

## 2. 学習資料
- React - The Complete Guide 2024
- 自習プロジェクト : <a href="https://github.com/thelight0804/World-weather-React">全世界の都市の天気提供サービス</a>
